Regras de uso das fotos

  • → Sempre usar object-cover object-top para manter o rosto centralizado.
  • → Aplicar gradiente from-surface-dark to-transparent no rodapé da foto para transição suave.
  • → Proporção padrão dos cards de equipe: aspect-ratio 3/4 (retrato).
  • → Tags de especialidade: borda aço, texto aço, fundo transparente — nunca filled.

Regras de Implementação

O que FAZER (Sempre)

  • Usar Cormorant Garant apenas para Display (H1/H2 e citações em destaque). Exige peso 500/600/700. Nunca use para parágrafos corridos.
  • Usar Lora para todo o corpo de texto (parágrafos, descrições, leads). Transmite elegância e legibilidade.
  • O azul-marinho #1C2E5C é a cor primária — use para botões, títulos sobre fundo claro e links ativos.
  • O cinza-aço #8090A8 é o único acento da paleta — use para ícones, labels, bordas de destaque e elementos secundários.
  • Aplicar prefers-reduced-motion no CSS global para desativar animações complexas se o usuário solicitar.
  • Manter área de clique mínima de 48x48px em botões mobile.
  • Intercalar seções claras com seções dark (.section-card.dark) para criar ritmo visual sem perder a sobriedade institucional.

O que NÃO FAZER (Nunca)

  • Nunca usar dourado ou amarelo em nenhum elemento. A paleta Wallach é navy + aço + branco. Nenhuma cor vibrante.
  • Nunca remover o border-radius: 1.5rem nem grudar a .section-card na borda da tela no Desktop. O fundo escuro #0D1829 deve sempre aparecer nas laterais.
  • Nunca aplicar gv-fade-slide-in em todos os elementos simultaneamente. Use as classes .delay-* para criar staggering visual.
  • Nunca usar verde, vermelho ou outras cores vibrantes fora dos estados de sucesso/erro em formulários.
  • Nunca usar Cormorant Garant em peso abaixo de 400 em telas pequenas — a fonte fica ilegível em tamanhos pequenos com peso leve.
